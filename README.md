# 💳 Credit Card Transaction Report (Power BI)
This project presents a comprehensive Credit Card Transaction Analytics Dashboard built using Power BI to analyze financial performance, customer behavior, and spending patterns. The dashboard provides a 360-degree view of transaction data, enabling stakeholders to monitor revenue streams, identify high-value customer segments, and optimize business strategies.

# 🔍 Overview
The dashboard highlights key financial KPIs including Total Revenue (55.3M), Interest Earned (7.8M), Total Transaction Amount, and Total Transaction Count, It is designed with a focus on financial insights, segmentation analysis, and trend evaluation, ensuring both operational and strategic decision-making.
The UI follows a structured card-based layout with consistent color theming, improving readability and executive usability.

# 📊 Key Insights & Analysis
- Revenue by Card Category:
    - Blue cards contribute the majority of revenue (~46M), followed by Silver, Gold, and Platinum, indicating a strong reliance on standard-tier customers.
- Customer Job Segmentation:
    - Business professionals and white-collar customers generate the highest revenue, highlighting key target segments for financial products.
- Expenditure Analysis:
    - Major spending categories include Bills, Entertainment, Fuel, and Grocery, providing insights into customer spending behavior and lifestyle patterns.
- Education Level Impact:
    - Customers with Graduate and High School education levels contribute significantly to revenue, indicating potential demographic targeting opportunities.
- Quarterly Performance Trends:
    - Q3 shows peak revenue and transaction count, while Q4 indicates a slight decline, helping identify seasonal trends in customer spending.
- Card-Level Financial Summary:
    - Detailed table analysis of revenue, transaction amount, and interest earned across card categories supports granular financial evaluation.

# 🛠️ Technical Implementation
- Designed a multi-layered data model integrating transactional and customer datasets
- Developed optimized DAX measures for financial KPIs such as revenue, interest earned, and transaction metrics
- Implemented interactive slicers (Gender, Week Start Date, Quarter) for dynamic filtering
- Utilized a variety of visuals:
    - KPI Cards for high-level metrics
    - Bar charts for segmentation analysis
    - Combo charts for quarterly trends
    - Table visual for detailed breakdown
- Applied data storytelling principles to guide users from summary metrics to detailed insights

# 🎯 Business Value

This dashboard enables organizations to:
- Monitor financial performance and transaction trends
- Identify high-value customer segments and spending behaviors
- Optimize credit card offerings and marketing strategies
- Analyze seasonal patterns in transactions
- Improve revenue generation through data-driven insights
